# Cucumber Homework Checklist

- [x] Maven project uses Cucumber, JUnit 4, and Selenide.
- [x] Folder structure includes `features`, `stepdefinitions`, `pages`, and `runners`.
- [x] At least three distinct `.feature` files exist.
- [x] At least one Scenario Outline exists.
- [x] At least one Data Table step exists.
- [x] Login success and login failure are covered.
- [x] Creating a new feed post is covered.
- [x] Searching for a user and opening that user's profile is covered.
- [x] Page Object Model is used.
- [x] Step definitions do not contain locators or direct Selenide selectors.
- [x] Hooks handle browser setup and teardown.
- [x] Cucumber HTML report is configured.
- [x] `mvn test` passes locally.
- [x] `target/cucumber-report.html` is generated.
